ASP.Net MVC

The StructureMap team recommends using the StructureMap.MVC5 nuget for integration into ASP.Net MVC applications. This nuget implements the nested container per HTTP request pattern that we recommend.

StructureMap.MVC5 is distributed as a Nuget at http://www.nuget.org/packages/StructureMap.MVC5 and can be installed from the command line with either:

ripple install StructureMap.MVC5 -p {your project name}

- or -

PM> Install-Package StructureMap.MVC5
